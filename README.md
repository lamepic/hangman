# Hangman

Hangman game made with react

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

```
Node
```

### Installing

```
clone repo into directory
```

```
cd into directory
```

```
npm install
```

```
npm start
```
